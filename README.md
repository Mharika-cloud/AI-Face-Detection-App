# 😊 AI Face Detection App

## 📌 Project Overview

This project was developed as part of the KODBUD AI Internship.

The application detects human faces from images using Computer Vision techniques and OpenCV.

## 🛠️ Technologies Used

* Python
* Google Colab
* OpenCV
* Computer Vision

## ✨ Features

* Upload image
* Detect human faces
* Draw bounding boxes around faces
* Display the number of faces detected
* Image processing using AI
